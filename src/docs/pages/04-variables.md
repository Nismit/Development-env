title: Defined Variables 
---

The variables are just defined in `_variable.scss`. It should be editable by each projects.

## Break Points

`xs`: 640
`sm`: 768
`md`: 1024
`lg`: 1200
`xl`: 1400

### Usage
`map-get($_bp-map, 'md')` -> `1024`
